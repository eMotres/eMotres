
# eMotres - Industrial Propulsion Solutions

## Overview

eMotres is a fictional company that specializes in high-performance electric motors for industrial applications, specifically for heavy-lift drones and eVTOLs. This Next.js application serves as the company's marketing and informational website.

## Design & Features

### Visual Design

*   **Color Palette:** The design uses a combination of dark gray, white, and a vibrant orange accent color (`#F97316`) to create a modern and industrial feel.
*   **Typography:** The site uses the "Inter" font, with a clear hierarchy of font sizes and weights to guide the user's attention.
*   **Layout:** The layout is clean and spacious, with a responsive design that adapts to different screen sizes. A container class is used to maintain a consistent width for the content.

### Features

*   **Hero Section:** A prominent hero section with a background image and a clear value proposition.
*   **Key Technologies:** A section that highlights the key features and technologies of the electric motors, presented in a card-based layout.
*   **Propulsion Matrix:** A table that displays the specifications of the different motor models.
*   **Navbar:** A responsive navigation bar with links to other pages.
*   **Footer:** A simple footer with copyright information.
